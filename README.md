# Embedded Line-Following Robot

Developed an embedded system for autonomous line following using Arduino-based hardware. The system integrates sensor input, PID control, and real-time interaction via a Node-RED interface.

## System Development Stages

### 1. PID Control System
Initial implementation of autonomous line following using infrared sensors and PID motor control for stable path tracking.

### 2. Node-RED Interface Integration
Extended the system to include real-time monitoring and interaction via a Node-RED dashboard.

### 3. Object Detection Extension
Explored computer vision techniques using OpenCV to detect and respond to visual inputs.

## Features
- Infrared sensor-based line detection  
- PID motor control for improved path stability  
- Master-slave communication architecture  
- Node-RED dashboard for real-time monitoring and interaction  
- Object detection extension using OpenCV  

## Technologies
- C / C++  
- Arduino  
- Node-RED  
- OpenCV (basic)
